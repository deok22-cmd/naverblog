📊 스프레드시트GOOGLEFINANCE투자 대시보드

주식 앱을 10개 깔아두고 매일 포트폴리오를 수동으로 업데이트하고 계신가요? 구글 스프레드시트의 GOOGLEFINANCE 함수는 그 모든 번거로움을 한 번에 해결해줍니다. 함수 하나로 실시간 주가, 환율, 시가총액, 52주 최고가·최저가, 심지어 과거 가격 차트 데이터까지 시트 안으로 불러올 수 있습니다.

이 가이드에서는 GOOGLEFINANCE 함수의 기본 문법부터 실전 투자 대시보드 구축까지, 단계별로 상세하게 설명합니다. 코딩 지식이 전혀 없어도 이 가이드를 따라오면 나만의 자동 업데이트 투자 현황판을 완성할 수 있습니다.

함수명GOOGLEFINANCE
지원 플랫폼Google Sheets (구글 스프레드시트)

비용무료 (Google Sheets 사용자라면 추가 비용 없음)

📐 GOOGLEFINANCE 기본 문법 완전 해부

GOOGLEFINANCE 함수의 전체 문법은 다음과 같습니다.

=GOOGLEFINANCE(티커, [속성], [시작일], [종료일_또는_기간], [간격])
각 인수의 역할을 하나씩 살펴보겠습니다.

① 티커 (필수)
조회할 종목의 티커 코드입니다. 국내 주식은 거래소 코드를 앞에 붙여야 합니다. 예를 들어 삼성전자는 "KRX:005930", 애플은 "AAPL", 테슬라는 "TSLA"입니다. 미국 주식은 거래소 코드 없이 티커만 입력해도 인식됩니다.

② 속성 (선택, 기본값: "price")
조회할 데이터의 종류입니다. 생략하면 현재가(price)를 반환합니다. 지원하는 속성 목록은 아래 섹션에서 상세히 설명합니다.

③ 시작일 (선택)
과거 데이터 조회 시 사용합니다. DATE(2026,1,1) 또는 "2026-01-01" 형식으로 입력합니다.

④ 종료일_또는_기간 (선택)
과거 데이터의 끝 날짜 또는 일수(숫자)를 입력합니다. TODAY() 함수와 조합하면 오늘까지의 데이터를 자동으로 가져올 수 있습니다.

⑤ 간격 (선택)
과거 데이터의 조회 단위입니다. "DAILY"(일별) 또는 "WEEKLY"(주별)를 사용합니다.

🏷️ 핵심 속성(attribute) 파라미터 총정리

🇰🇷 국내 주식 조회 — KRX 거래소 종목 코드 사용법

한국 주식을 조회할 때는 반드시 거래소 코드 KRX: 를 티커 앞에 붙여야 합니다. 종목 코드는 6자리 숫자이며 한국거래소(KRX)에서 부여됩니다.

국내 주요 종목 티커 코드 예시

삼성전자: =GOOGLEFINANCE("KRX:005930") SK하이닉스: =GOOGLEFINANCE("KRX:000660") NAVER: =GOOGLEFINANCE("KRX:035420") 카카오: =GOOGLEFINANCE("KRX:035720") LG에너지솔루션: =GOOGLEFINANCE("KRX:373220") 현대차: =GOOGLEFINANCE("KRX:005380") 셀트리온: =GOOGLEFINANCE("KRX:068270")

국내 종목 코드를 모를 경우, 네이버 금융이나 한국거래소 공식 사이트에서 종목명으로 검색하여 6자리 코드를 확인한 뒤 사용합니다.

💡 실전 팁: 셀 참조로 티커 코드 관리하기
티커 코드를 함수 안에 직접 쓰는 것보다, A1셀에 "KRX:005930"을 입력하고 =GOOGLEFINANCE(A1,"price") 와 같이 셀 참조를 사용하면 종목을 바꿀 때 훨씬 편리합니다. 티커 코드 목록을 별도 열에 만들어 관리하면 대시보드 확장도 쉽습니다.

💱 환율 실시간 조회 — CURRENCY 쌍 지정법

환율 데이터는 CURRENCY:기준통화종결통화 형식의 티커를 사용합니다. 예를 들어 달러/원 환율은 "CURRENCY:USDKRW"입니다.

달러/원 (USD → KRW): =GOOGLEFINANCE("CURRENCY:USDKRW") 유로/원 (EUR → KRW): =GOOGLEFINANCE("CURRENCY:EURKRW") 달러/엔 (USD → JPY): =GOOGLEFINANCE("CURRENCY:USDJPY") 달러/위안 (USD → CNY): =GOOGLEFINANCE("CURRENCY:USDCNY") 파운드/달러 (GBP → USD): =GOOGLEFINANCE("CURRENCY:GBPUSD")

예를 들어 미국 주식 포트폴리오를 원화로 환산하여 관리하고 싶다면, 달러 기준 평가액 셀에 GOOGLEFINANCE("CURRENCY:USDKRW") 값을 곱해주기만 하면 자동으로 원화 환산이 이루어집니다.

실전 예제: 원화 환산 포트폴리오 수식

A2: AAPL (티커) B2: =GOOGLEFINANCE(A2) (현재가, 달러) C2: 10 (보유 수량) D2: =B2*C2 (달러 평가액) E1: =GOOGLEFINANCE("CURRENCY:USDKRW") (현재 달러/원 환율) E2: =D2*$E$1 (원화 환산 평가액, 자동 갱신)

📈 과거 가격 데이터 — 히스토리컬 차트 구현

GOOGLEFINANCE는 과거 데이터를 조회하는 것도 가능합니다. 시작일과 종료일을 지정하면 해당 기간의 OHLCV(시가·고가·저가·종가·거래량) 데이터가 범위 형식으로 반환됩니다.

특정 기간 종가 데이터: =GOOGLEFINANCE("KRX:005930", "close", DATE(2026,1,1), TODAY()) 최근 90일 일별 종가: =GOOGLEFINANCE("AAPL", "close", TODAY()-90, TODAY(), "DAILY") 주별 데이터: =GOOGLEFINANCE("TSLA", "close", DATE(2025,1,1), TODAY(), "WEEKLY")

과거 데이터를 불러오면 날짜·종가 두 열로 구성된 배열이 아래로 자동 확장됩니다. 이 데이터를 선택 후 차트 삽입 기능을 이용하면 가격 추이 라인 차트를 쉽게 만들 수 있습니다.

⚠️ 주의: 과거 데이터 위치 지정
과거 데이터는 함수를 입력한 셀을 시작으로 아래쪽으로 자동 확장됩니다. 함수 셀 아래에 데이터가 있으면 "#REF!" 오류가 발생합니다. 반드시 아래쪽 행들이 비어 있는 빈 공간에 입력하세요.

🏦 뮤추얼펀드 & ETF 데이터 조회

GOOGLEFINANCE는 주식 외에도 뮤추얼펀드와 ETF 데이터도 조회 가능합니다. 미국 ETF는 일반 주식과 동일한 방식으로 티커를 입력합니다.

SPY (S&P500 ETF): =GOOGLEFINANCE("SPY") QQQ (나스닥100 ETF): =GOOGLEFINANCE("QQQ") TIGER 200 (국내 ETF): =GOOGLEFINANCE("KRX:102110") KODEX 반도체: =GOOGLEFINANCE("KRX:091160")

뮤추얼펀드는 펀드 심볼(일반적으로 5자리 문자 + X)을 사용합니다. 예를 들어 Vanguard Total Stock Market Fund는 =GOOGLEFINANCE("MUTF:VTSAX")로 조회합니다.

🖥️ 나만의 투자 대시보드 완전 구축 — 실전 예제

이 섹션에서는 실제로 사용할 수 있는 투자 대시보드의 구조와 수식을 단계별로 설명합니다.

① 시트 구조 설계

[A열] 종목명 (예: 삼성전자, 애플, 테슬라)
[B열] 티커 코드 (예: KRX:005930, AAPL, TSLA)
[C열] 현재가 =GOOGLEFINANCE(B2)
[D열] 등락률(%) =GOOGLEFINANCE(B2,"changepct")
[E열] 52주 최고가 =GOOGLEFINANCE(B2,"high52")
[F열] 52주 최저가 =GOOGLEFINANCE(B2,"low52")
[G열] 평균 매수가 (수동 입력)
[H열] 보유 수량 (수동 입력)
[I열] 매수 총액 =G2*H2
[J열] 현재 평가액 =C2*H2
[K열] 수익률(%) =(J2-I2)/I2*100

② 조건부 서식으로 수익/손실 색상 표시

K열(수익률)에 조건부 서식을 적용합니다. 양수(수익)는 초록색, 음수(손실)는 빨간색 배경으로 자동 표시되게 하면 한눈에 포트폴리오 현황을 파악할 수 있습니다.

K열 선택 → 서식 메뉴 → 조건부 서식 → "다음보다 큰" → 0 → 초록 배경 적용 / "다음보다 작은" → 0 → 빨간 배경 적용

③ SPARKLINE으로 셀 안에 미니 차트 삽입

=SPARKLINE(GOOGLEFINANCE(B2,"close",TODAY()-30,TODAY(),"DAILY"),{"charttype","line";"color","#1565c0";"linewidth",2})
이 수식은 최근 30일간의 종가 데이터를 가져와 셀 안에 파란색 라인 차트로 그려줍니다. 테이블 구조를 유지하면서 각 종목의 단기 추세를 직관적으로 볼 수 있습니다.

⚠️ GOOGLEFINANCE 사용 시 주의사항 및 제약

⚠️ 반드시 알아야 할 제약사항
1. 약 20분 지연 데이터: GOOGLEFINANCE는 실시간 데이터가 아닙니다. 증권사 HTS의 실시간 데이터와 최대 20분 차이가 날 수 있으므로, 초단기 트레이딩 판단 자료로 사용하면 안 됩니다.
2. 일부 한국 거래소 속성 미지원: KRX 종목의 경우 "pe", "eps" 등 일부 속성이 반환되지 않을 수 있습니다. N/A 오류가 발생하면 IFERROR 함수로 감싸 처리하세요.
3. 상장폐지 종목 오류: 상장폐지되거나 합병된 종목은 오류를 반환합니다.
4. 주말·공휴일 처리: 주말이나 공휴일에는 마지막 거래일 가격을 반환합니다.

IFERROR로 오류 처리하기

=IFERROR(GOOGLEFINANCE("KRX:005930","pe"), "N/A")
지원되지 않는 속성에 IFERROR를 적용하면 오류 대신 "N/A" 텍스트가 표시되어 대시보드가 깔끔하게 유지됩니다.

🔗 SPARKLINE과 조합 — 셀 내 미니 차트 완성판

SPARKLINE 함수와 GOOGLEFINANCE를 결합하면 별도 차트 없이도 셀 안에 가격 추이를 시각화할 수 있습니다. 이 조합은 대시보드의 시각적 완성도를 크게 높여줍니다.

라인 차트 (최근 60일): =SPARKLINE( GOOGLEFINANCE("AAPL","close",TODAY()-60,TODAY(),"DAILY"), {"charttype","line";"color","#1e88e5";"linewidth",2} ) 컬럼 차트 (최근 30일 거래량): =SPARKLINE( GOOGLEFINANCE("AAPL","volume",TODAY()-30,TODAY(),"DAILY"), {"charttype","column";"color","#43a047"} ) 고가/저가 분포 막대: =SPARKLINE( {GOOGLEFINANCE("AAPL","low52"), GOOGLEFINANCE("AAPL","price"), GOOGLEFINANCE("AAPL","high52")}, {"charttype","bar";"color1","#ef5350";"color2","#1e88e5"} )

이 수식들을 각 종목 행의 별도 열에 배치하면, 포트폴리오 테이블 안에서 각 종목의 최근 가격 흐름을 직관적으로 비교할 수 있습니다. 별도의 시트 이동이나 앱 전환 없이 한 화면에서 모든 정보가 자동 갱신됩니다.

📊 실전 활용 사례 — 환율 모니터링 대시보드

GOOGLEFINANCE는 주식 투자자뿐 아니라 해외 거래나 유학·여행 준비 중인 분들에게도 유용합니다. 주요 통화 환율을 한 시트에서 모니터링하는 간단한 환율 대시보드를 구성하는 예시입니다.

A1: "통화쌍" B1: "현재 환율" C1: "어제 대비" A2: USD/KRW B2: =GOOGLEFINANCE("CURRENCY:USDKRW") A3: EUR/KRW B3: =GOOGLEFINANCE("CURRENCY:EURKRW") A4: JPY/KRW B4: =GOOGLEFINANCE("CURRENCY:JPYKRW") A5: CNY/KRW B5: =GOOGLEFINANCE("CURRENCY:CNYKRW") A6: GBP/KRW B6: =GOOGLEFINANCE("CURRENCY:GBPKRW")

해당 시트를 구글 스프레드시트 앱으로 스마트폰에서도 열 수 있으므로, 별도 환율 앱 없이도 한눈에 주요 환율을 확인하는 개인 환율 모니터가 완성됩니다.
