REGEXEXTRACT & REGEXREPLACE 완벽 가이드
정규표현식으로 데이터 클렌징 끝내기 2026

엑셀이나 구글 스프레드시트를 쓰다 보면 반드시 마주치는 상황이 있습니다. 셀 하나에 이름, 전화번호, 이메일이 한꺼번에 뭉쳐 있거나, 날짜 형식이 제멋대로 섞여 있거나, 필요 없는 문자들이 끼어 있는 지저분한 데이터입니다. 이 문제를 가장 강력하게 해결하는 도구가 바로 REGEXEXTRACT와 REGEXREPLACE 함수입니다. 정규표현식(Regular Expression)이라는 패턴 언어를 사용하여 어떤 형태의 문자열도 원하는 방식으로 추출하고 변환할 수 있습니다. 오늘 이 두 함수를 완벽히 정복하면, 데이터 클렌징 작업의 90%는 자동화할 수 있습니다.

📌 정규표현식(Regex)이란 무엇인가

정규표현식(Regular Expression, 줄여서 Regex)은 문자열에서 특정 패턴을 찾거나 교체하기 위한 언어입니다. 마치 검색창에서 "단어 포함"으로 검색하는 것보다 훨씬 정밀하게, 패턴 자체를 정의하여 문자열을 다루는 기술입니다.

예를 들어, "연락처: 010-1234-5678 / 이메일: hong@gmail.com"이라는 문장에서 전화번호만 뽑아내고 싶다면, 정규표현식으로 "숫자 3자리 - 숫자 4자리 - 숫자 4자리 패턴"을 정의하면 자동으로 추출해 줍니다.

구글 스프레드시트는 Google의 RE2 정규표현식 엔진을 사용합니다. 파이썬이나 자바스크립트의 정규표현식과 유사하나, 일부 기능(예: lookbehind 단언)이 지원되지 않으니 참고하세요.

🔍 필수 정규표현식 패턴 사전

REGEXEXTRACT와 REGEXREPLACE를 사용하기 전에, 자주 쓰이는 기본 패턴들을 먼저 익혀두세요. 이 패턴들은 실무에서 80% 이상의 상황을 커버합니다.

패턴의미예시
\d숫자 하나 (0~9)\d\d\d = 숫자 3자리
\d+숫자 1개 이상 연속\d+ = 1234, 99, 0
\d{3}숫자 정확히 3개\d{3} = 010, 123
\d{3,4}숫자 3~4개전화번호 중간/끝 부분
[A-Za-z]영문자 한 글자[A-Za-z]+ = 영단어
[가-힣]한글 한 글자[가-힣]+ = 한글 단어
.임의의 한 문자.+ = 모든 문자열
^문자열 시작^서울 = '서울'로 시작
$문자열 끝\.com$ = .com으로 끝남
()그룹 지정 (캡처)(\d+)원 = 숫자 부분 추출
|OR 조건서울|부산 = 서울 또는 부산
\s공백 문자\s+ = 공백 여러 개
\w단어 문자(문자+숫자+_)\w+ = 단어

📤 REGEXEXTRACT — 원하는 패턴만 쏙 뽑기

REGEXEXTRACT는 셀에서 정규표현식 패턴에 맞는 첫 번째 문자열을 추출하는 함수입니다.

기본 문법:
=REGEXEXTRACT(텍스트, 정규표현식)
패턴이 일치하지 않으면 오류(#N/A)가 반환됩니다. 오류 방지를 위해 IFERROR로 감싸는 것을 권장합니다.

예제 1: 전화번호 추출
A1 셀에 "고객명: 홍길동 / 연락처: 010-1234-5678 / 부서: 영업1팀" 이 있을 때
=REGEXEXTRACT(A1, "\d{3}-\d{3,4}-\d{4}")
결과: 010-1234-5678
예제 2: 이메일 주소 추출
A1 셀에 "담당: kim.sales@company.co.kr" 이 있을 때
=REGEXEXTRACT(A1, "[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}")
결과: kim.sales@company.co.kr
예제 3: 괄호 안 텍스트만 추출
A1 셀에 "홍길동(영업팀장)" 이 있을 때
=REGEXEXTRACT(A1, "\(([^)]+)\)")
결과: (영업팀장) — 괄호 제거를 원하면 캡처 그룹 활용
=REGEXEXTRACT(A1, "\(([^)]+)\)", 1)
결과: 영업팀장
예제 4: 숫자(금액)만 추출
A1 셀에 "결제금액: 35,000원" 이 있을 때
=REGEXEXTRACT(A1, "[\d,]+")
결과: 35,000
예제 5: 날짜 형식 추출 (YYYY-MM-DD)
A1 셀에 "등록일: 2026-05-06 (수)" 이 있을 때
=REGEXEXTRACT(A1, "\d{4}-\d{2}-\d{2}")
결과: 2026-05-06

🔄 REGEXREPLACE — 패턴 찾아서 바꾸기

REGEXREPLACE는 셀에서 정규표현식 패턴에 맞는 부분을 다른 텍스트로 교체하는 함수입니다.

기본 문법:
=REGEXREPLACE(텍스트, 정규표현식, 교체할_텍스트)
[REGEXREPLACE로 데이터 정제 전후 비교] / AI 생성 이미지
예제 1: 전화번호 형식 통일 (하이픈 제거)
A1 셀에 "010-1234-5678"이 있을 때 하이픈 없애기
=REGEXREPLACE(A1, "-", "")
결과: 01012345678
예제 2: 연속된 공백 → 단일 공백으로 정리
A1 셀에 "홍길동   영업팀"처럼 공백이 여러 개일 때
=REGEXREPLACE(A1, "\s+", " ")
결과: 홍길동 영업팀
예제 3: 특수문자 전부 제거
A1 셀에 "가격!@#$: 35,000원!!!"이 있을 때
=REGEXREPLACE(A1, "[^가-힣A-Za-z0-9\s,.]", "")
결과: 가격 35,000원
예제 4: 날짜 형식 변환 (2026.05.06 → 2026-05-06)
A1 셀에 "2026.05.06"이 있을 때
=REGEXREPLACE(A1, "\.", "-")
결과: 2026-05-06
예제 5: 개인정보 마스킹 처리
A1 셀에 "010-1234-5678"에서 뒷 4자리 마스킹
=REGEXREPLACE(A1, "\d{4}$", "****")
결과: 010-1234-****

🏆 실무 활용 시나리오 — 이 상황이라면 이 패턴!

현업에서 반드시 마주치는 데이터 처리 상황별 최적 REGEX 공식을 정리했습니다.

시나리오 1: 주소에서 시/도만 추출
고객 주소가 "서울특별시 강남구 테헤란로 123"일 때 '서울특별시'만 추출
=REGEXEXTRACT(A1, "^[^\s]+")
결과: 서울특별시

시나리오 2: URL에서 도메인만 추출
"https://www.example.co.kr/page/detail"에서 도메인만
=REGEXEXTRACT(A1, "https?://([^/]+)")
결과: www.example.co.kr

시나리오 3: 숫자 뒤 단위 텍스트 제거
"35,000원", "120개", "15km" 등에서 숫자와 쉼표만 남기기
=REGEXREPLACE(A1, "[^0-9,]", "")
결과: 35,000 / 120 / 15

시나리오 4: 영문 이메일 도메인 부분만 추출
"hong@gmail.com"에서 "@" 뒤만 추출
=REGEXEXTRACT(A1, "@(.+)$")
결과: gmail.com

⚠️ 자주 실수하는 패턴과 해결법

⚠️ 실수 1: 특수문자를 이스케이프하지 않음
`.`, `*`, `+`, `(`, `)`, `[`, `]` 등 정규표현식에서 특별한 의미를 가진 문자를 그대로 검색하려면 앞에 백슬래시(\)를 붙여야 합니다.
예: 점(.)을 찾을 때 → \. / 괄호를 찾을 때 → \(, \)
⚠️ 실수 2: REGEXEXTRACT에서 일치 없을 때 오류
패턴이 일치하지 않으면 #N/A 오류가 발생합니다. 실무에서는 항상 IFERROR로 감싸세요.
=IFERROR(REGEXEXTRACT(A1, "\d{3}-\d{4}-\d{4}"), "전화번호 없음")
⚠️ 실수 3: 대소문자 구분
기본적으로 REGEX 함수는 대소문자를 구분합니다. 대소문자 무관하게 검색하려면 패턴 앞에 (?i)를 추가하세요.
=REGEXEXTRACT(A1, "(?i)gmail")
이 패턴은 "Gmail", "GMAIL", "gmail" 모두 일치합니다.
💡 팁: REGEXMATCH로 조건 확인 후 사용
추출 전에 REGEXMATCH 함수로 패턴이 존재하는지 먼저 확인하면 더욱 안전합니다.
=IF(REGEXMATCH(A1, "\d{3}-\d{4}-\d{4}"), REGEXEXTRACT(A1, "\d{3}-\d{4}-\d{4}"), "해당없음")

🔗 ARRAYFORMULA와 결합하여 전체 열 처리

ARRAYFORMULA와 결합하면 하나의 수식으로 전체 열의 데이터를 한꺼번에 변환할 수 있어 실무 효율이 극대화됩니다.

전체 A열 전화번호 하이픈 제거 — 한 번에
=ARRAYFORMULA(IF(A2:A="", "", REGEXREPLACE(A2:A, "-", "")))
A2:A 전체에 대해 하이픈을 제거한 결과를 한 셀에서 모두 출력합니다.
전체 B열에서 이메일만 추출 — 한 번에
=ARRAYFORMULA(IFERROR(REGEXEXTRACT(B2:B, "[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}"), "없음"))
B열 전체에서 이메일 주소를 추출하되, 없는 경우 "없음"으로 표시합니다.
ARRAYFORMULA와의 조합은 수천 행의 데이터를 처리할 때 특히 강력합니다. 개별 셀마다 수식을 복사하는 수작업 없이, 한 줄의 수식으로 전체 데이터 클렌징이 완료됩니다.

📊 REGEXEXTRACT vs SPLIT vs LEFT/MID/RIGHT 비교

함수장점단점적합한 상황
REGEXEXTRACT복잡한 패턴 처리 가능, 위치 무관Regex 문법 학습 필요이메일, 전화번호, 날짜 등 패턴이 있는 데이터
SPLIT간단하고 직관적구분자가 일정해야 함콤마, 슬래시 등 구분자로 나뉜 데이터
LEFT/MID/RIGHT위치 기반으로 확실위치가 고정되어야 함앞 5자리, 특정 위치의 문자열
데이터의 패턴이 불규칙하거나 위치가 고정되지 않은 경우, REGEXEXTRACT가 압도적으로 유리합니다. 반면 구분자가 명확하다면 SPLIT이 더 간결할 수 있으니 상황에 맞게 선택하세요.

🛠️ 오늘 바로 쓸 수 있는 REGEX 공식 모음집

복사해서 바로 활용할 수 있는 완성된 공식들입니다.

한국 전화번호 추출:
=IFERROR(REGEXEXTRACT(A1, "0\d{1,2}-\d{3,4}-\d{4}"), "")
이메일 추출:
=IFERROR(REGEXEXTRACT(A1, "[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,}"), "")
한글만 추출:
=IFERROR(REGEXEXTRACT(A1, "[가-힣]+"), "")
숫자만 추출:
=IFERROR(REGEXEXTRACT(A1, "\d+"), "")
영문자만 추출:
=IFERROR(REGEXEXTRACT(A1, "[A-Za-z]+"), "")
날짜(YYYY-MM-DD) 추출:
=IFERROR(REGEXEXTRACT(A1, "\d{4}-\d{2}-\d{2}"), "")
URL 추출:
=IFERROR(REGEXEXTRACT(A1, "https?://[^\s]+"), "")
특수문자 모두 제거:
=REGEXREPLACE(A1, "[^가-힣A-Za-z0-9\s]", "")
앞뒤 공백 및 연속 공백 정리:
=TRIM(REGEXREPLACE(A1, "\s+", " "))

❓ REGEX 함수 자주 묻는 질문

Q. REGEXEXTRACT로 여러 개의 일치값을 모두 추출할 수 있나요?
A. 기본 REGEXEXTRACT는 첫 번째 일치값만 반환합니다. 모든 일치값을 추출하려면 Google Apps Script를 활용하거나, 데이터 전처리 후 여러 REGEXEXTRACT를 조합해야 합니다.

Q. 한글 자음/모음도 정규표현식으로 처리할 수 있나요?
A. 가능합니다. 한글 자음은 [ㄱ-ㅎ], 모음은 [ㅏ-ㅣ], 완성형 한글은 [가-힣]으로 표현합니다.

Q. REGEXREPLACE에서 교체 텍스트에 캡처 그룹을 사용할 수 있나요?
A. 네, 가능합니다. 교체 텍스트에서 \1, \2 등으로 캡처된 그룹을 참조할 수 있습니다.
예: "홍길동(010-1234-5678)"에서 "(전화번호: 010-1234-5678)"로 변환
=REGEXREPLACE(A1, "([가-힣]+)\((\d{3}-\d{4}-\d{4})\)", "\1(전화번호: \2)")
