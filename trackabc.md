**운영 룰**:
1. (권장) 동일 시즌 키워드(예: "장미축제", "수국") 다수 발행 시 자기잠식 가능성 있음 — 가능하면 분산 발행 권장. 단 검색 진입 경로가 다양하므로 절대 금지는 아님.
2. (필수) 트랙 A·C는 일반어 단독 제목 금지 — 반드시 **동/구/마을 단위 고유명사** 포함
3. (필수) 트랙 B는 정확한 일정(YYYY.MM.DD~MM.DD)을 본문 내에 명시 의무. **회차 표기("제N회")는 사용하지 않는다** — 추정 회차가 실제와 빈번히 어긋나 신뢰도 손상 위험이 크다(2026-05-14 양구 곰취·강진 청자 사례).
4. (필수) 스프레드시트·레시피 카테고리는 신규 발행 중단 (기존 `sub_topic_tracker.md` 동결)
5. (필수) 트랙 B 발굴 시점은 **행사 시작일 기준 D-7 ~ D-35 범위만 허용**. D-36 이상 멀리 있는 행사는 큐에 미리 적재하지 않고, 해당 시점에 별도 발굴 작업으로 처리.

## 6. 주제 추가 표준 절차 (Topic Pool Replenishment) — 상세

`Naverblog.md` 02-bis조의 운영 골격에 대한 상세 가이드. 신규 주제를 발굴할 때 본 절차를 따른다.

### 6.1 발굴 트리거 — 3가지만 인정

| 트리거 | 감지 방법 | 작업 범위 | 운영 빈도 |
|---|---|---|---|
| **T1. 큐 부족** | `국내여행지.md` '작성대기중'을 트랙별로 카운트, 임계치 이하면 발동 | 부족 트랙만 보충 | 비정기 |
| **T2. B트랙 시즌 윈도우 진입** | 매주 월요일 기준, D-7~D-35 범위에 들어오는 시즌 행사 점검 | B트랙만 신규 발굴 | 매주 1회 |
| **T3. 사용자 즉흥 추가** | 사용자가 채팅으로 명소·축제 명시 | 단건 또는 묶음 | 비정기 |

이 외 사유(예: "재미있어 보여서 추가")로는 풀을 건드리지 않는다.

### 6.2 표준 5단계 절차

```
[1] trackabc.md에 후보 등재
    ├─ 신규 섹션 또는 기존 표 하단에 추가
    ├─ 트랙(A/B/C) 분류 명시
    ├─ 일정(B는 행사일 YYYY.MM.DD~MM.DD, A·C는 시즌성 라벨)
    ├─ 검색 경쟁 가설(C 트랙만, 약/중/강)
    └─ 발행 포인트 1줄

[2] 검증·토론
    ├─ B 트랙: WebSearch로 회차 번호·시작일·종료일 일괄 팩트체크
    │   (지자체 공식 페이지 / 한국관광공사 기준)
    ├─ A·C 트랙: 동/구/마을 단위 고유명사 포함 여부 확인
    ├─ 사용자 검토 → 제외/수정 항목 지정
    └─ 트랙 분류 재조정(시즌성 있는 A → B로 등)

[3] 국내여행지.md 02 섹션으로 승격
    ├─ 일련번호 부여(현재 마지막 번호 +1부터)
    ├─ 트랙 컬럼 채움 (A/B/C)
    ├─ 상태 = "작성대기중"
    └─ 키워드·주요 기간·발행 포인트는 trackabc.md에서 그대로 복사

[4] trackabc.md에 "승격 완료" 마킹
    ├─ 해당 행 끝에 `→ 국내여행지.md #NNN~#NNN 등재(YYYY-MM-DD)` 표시
    └─ 또는 별도 "승격 완료" 섹션으로 이동

[5] git commit
    └─ 형식: "주제 풀 보충: 트랙 X N건 신규 등재 (#NNN~#NNN)"
```

### 6.3 트랙별 보충 주기 및 임계치

| 트랙 | 보충 주기 | 1회 발굴 건수 | 큐 안전 재고 | 비고 |
|---|---|---|---|---|
| **A. 수도권 마이크로** | 큐 5건 이하 시 (비정기) | 10~15건 | 10건 이상 | 에버그린, 시기 제약 없음 |
| **B. 시즌 D-7~D-35** | **매주 월요일** 정기 | 5~10건 | 7건 이상 | D-35 룰로 큐 회전이 빠름 |
| **C. 지방 니치** | 큐 3건 이하 시 (비정기) | 5건 | 5건 이상 | 검색 점유 가설 검증 필수 |

### 6.4 자동화 vs 사용자 결정 분리

| 단계 | 자동화 가능 | 사용자 결정 필요 |
|---|---|---|
| 큐 카운트 감지 | ✅ 스크립트 가능 | — |
| 후보 발굴 | ⚠️ Claude가 초안 작성 | ✅ 채택 여부 |
| 회차·일정 검증 | ✅ WebSearch 일괄 | — |
| 트랙 분류 | ⚠️ Claude가 초안 작성 | ✅ 재조정 |
| `국내여행지.md` 승격 | ✅ 스크립트 가능 | ⚠️ 승인 |
| git commit | ✅ 스크립트 가능 | — |
