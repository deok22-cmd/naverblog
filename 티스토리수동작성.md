# 티스토리 원고 수동 작성 가이드 (2026-05-20 신설)

> 자동 발행 일시 중단(`daily-run.ps1` `$TistorySuspended = $true`) 기간 동안 사용한다. 사용자가 *단건* 요청할 때만 작성.

## 트리거 (사용자 발화 예)
- "○○ 지역 ○○ 느낌으로 티스토리 원고 작성해줘"
- "△△ 여행 후기 티스토리에 올릴 거 1건 써줘"

## 입력 (사용자가 매번 지정 / 미지정 시 한 번만 묻는다)
- **지역·테마**: 필수
- **이미지 개수**: 사용자가 별도 지시 (기본 = 0개. 임의로 박지 않는다)
- **이미지 출처**: 사용자가 보유한 실제 여행 사진 파일 경로 — **AI 생성 이미지·placeholder 박스 절대 금지**
- **분량**: 별 지시 없으면 본문 약 5,000~7,000자 (h2 6~9개)
- **슬러그**: 미지정 시 `travel_<지역>_<테마>_2026`

## 출력
- 경로: `output_tistory/<YYMMDD>/<slug>.html` 1건만
- 네이버·인스타 등 **다른 채널 동봉 X**
- `국내여행지.md` 트래커 자동 마킹 **X** (수동 요청은 큐와 별개)
- git commit/push는 사용자 요청 시에만

## 작성 규칙 (자동 모드와 동일한 표준)
- 외곽 래퍼 div (`max-width: 780px; ... padding: 16px;`)
- Platinum h1 (`font-weight: 800; border-bottom: 5px solid #00796b;` + `text-align: center;`)
- h2 번호 prefix `01.` `02.` …, 인라인 teal 배경
- `<style>` 블록·`class=` 속성 **절대 금지** (인라인만)
- 표 행 배경 각 `<td>` 직접 (`#f1faf9`/`#fff`)
- 추천 박스(`🔗 같이 보면 좋은 글`) + 태그 칩 **10개**
- 어조: 존대말, 문단 사이 `<br><br>`

## 이미지 처리 (핵심 차이)
- 사용자가 N개 지정 시에만 N개 삽입. **임의로 3개를 박지 않는다.**
- 사용자가 0개 또는 미지정 시 이미지 없는 텍스트 원고로 작성.
- 삽입 시 표준 1줄 블록 (사용자 제공 경로 그대로):
  ```html
  <div style="margin:30px 0;text-align:center;"><img alt="…" src="<사용자 제공 경로>" style="max-width:100%;height:auto;border-radius:8px;"/><div style="font-size:0.9em;color:#666;margin-top:10px;font-style:italic;">…</div></div>
  ```
- **금지**: `📷 AI 이미지 생성 영역`·`프롬프트 복사하기`·`class="img-placeholder"`·`class="prompt-text"` 등 placeholder 박스 일체 (게이트 F3 위반).

## 검수
저장 후 권장 실행:
```
powershell -File .scripts/tistory-gate.ps1 -Day <YYMMDD>
```
F1~F8 + 중복 PASS면 안전 (수동 푸시는 사용자가 직접 또는 별도 요청 시).

## 자동 재개 절차
사용자가 *"티스토리 원고 자동작성 재개해줘"* 라고 요청하면 즉시 다음 3가지를 동시 복원:
1. `daily-run.ps1` 상단 `$TistorySuspended = $false`
2. `.scripts/daily-prompt.md` 큰 틀 줄·§5 헤더의 SUSPENDED 배너 제거
3. 본 파일 상단에 `> ※ 재개 완료(YYYY-MM-DD) — 본 가이드는 보존만, 자동 모드는 daily-prompt §5 사용.` 추가

다음 4시부터 자동 5건 미러 작성 재개.
